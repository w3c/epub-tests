Test passes (i.e., the Reading System supports playback of two consecutive XHTML Media documents, each with an associated MO file) if

  1. The next (second) page is properly displayed as an XHTML content
  2. The Reading System provides the user with a way to turn text reading on
  3. If turned on, the extract on the second page is read aloud by the Reading System
  4. After the full paragraph is read, the content automatically switches to the next portion of the text on the third page, while reading aloud continues without interruption.

The Reading System may highlight the text being read with a green background with the text grayed out; but this feature is not required for the test to pass.
