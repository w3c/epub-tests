The hreflang attribute must be ignored when processing the target of a link element

This test is only relevant if the reading system supports linked resources and, in particular, the description linked to (rather than embedded in) the package document is shown by the user interface. If that is not the case, the relevant entry in the implementation report should be set to null.

This epub's package document contains a link element to the detailed description of the test. The description itself provides the criteria for the test to pass.
