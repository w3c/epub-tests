Test passes (i.e., the Reading System plays an audio track from clipBegin to clipEnd) if

  1. The Reading System provides the user with a way to turn text reading on
  2. When turned on, the extract on the next page is read aloud by the Reading System
  3. The full text is read
  4. No additional text is read (even though the audio file is longer than the text)

The Reading System may highlight the text being read with a green background with the text grayed out; but this feature is not required for the test to pass.
