Test passes (i.e., the Reading System correctly handles the lack of a clipBegin attribute value) if

  1. The Reading System provides the user with a way to turn text reading on
  2. If turned on, the extract on the next page is read aloud by the Reading System
  3. The audio does not start at the beginning of the text but at the title of the publication, name of the reader, etc.

The Reading System may highlight the text being read with a green background with the text grayed out; but this feature is not required for the test to pass.
