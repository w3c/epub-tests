Test passes (i.e., the Reading System correctly falls back on Text-to-Speech when no audio file is present) if

  1. The Reading System provides the user with a way to turn text reading on
  2. If turned on, the extract on the next page is read aloud by the Reading System using Text-to-Speech

The Reading System may highlight the text being read with a green background with the text grayed out; but this feature is not required for the test to pass.
