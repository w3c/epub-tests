Description of the EPUB 3.3 tests.

Introduction

This is the repository for the the tests to validate the implementability of W3C's EPUB 3.3 specifications, specifically the core EPUB 3.3 (the specification for the EPUB format itself) and the EPUB Reading Systems 3.3 (the specification for applications that read and render EPUB files).

The development of the tests is a community effort (see the list of contributors). Everyone is welcome to contribute tests; please read the separate contribution guidelines if you are interested in doing so.

This document provides a short overview for each test, grouped into separate tables by test categories. The columns in the tables are as follows:

  1. The identification of each test, linked to the unzipped tests themselves in the github repository
  2. Title of the test
  3. Short description of the test
  4. References to the normative statements or sections in the EPUB 3.3 specification that this test is relevant to
  5. Pointer to the (consolidated) implementation status of the test in the separate implementation report file
