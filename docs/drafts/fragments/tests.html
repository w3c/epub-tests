Publication Resources

Id Title Description Specs Ref
confreq-rs-data-urls Data URL does not open in top-level context XHTML content contains an img element where the src attribute is given as a data URL. RS must display the image rather than opening it in a new top-level browsing context. (1) (2) ❐
confreq-rs-foreign_bad-fallback Manifest fallback with unsupported fallback This is a test of manifest fallbacks where both the spine item and the fallback are likely to be unsupported. The spine item is a DMG, with a fallback to a PSD file. (1) ❐
confreq-rs-foreign_image Manifest fallback with image An HTML content file contains a PSD image, with a manifest fallback to a PNG image. This tests fallbacks for resources that are not in the spine. (1) ❐
confreq-rs-foreign_json-spine Manifest fallback With JSON content document in spine This EPUB uses a JSON content file in the spine, with a manifest fallback to an HTML document. If the reading system does not support JSON, it should display the HTML. (1) ❐
confreq-rs-foreign_xml-spine Manifest fallback with XML in spine This EPUB uses an ordinary XML content file with mimetype application/xml in the spine, with a manifest fallback to an HTML document. If the reading system does not support XML, it should display the HTML. (1) ❐
confreq-rs-foreign_xml-suffix-spine Manifest fallback with XML with suffix in spine This EPUB uses an custom XML content file with mimetype application/dtc+xml in the spine, with a manifest fallback to an HTML document. If the reading system does not support XML, it should display the HTML. (1) ❐
confreq-rs-xml-extid XML external entity not resolved The XHTML content document contains a reference to an external entity. The reading system must not resolve it. (1) ❐
confreq-rs-xml-ns Invalid XML element name An XHTML element has an invalid name with two successive colons. The reading system must produce an error. (1) ❐
confreq-rs-xml-nval_invalid Invalid XML navigation document The spine XML is invalid. The reading system must not display the navigation document. (1) ❐
confreq-rs-xml-nval_unclosed Unclosed XML tag An XHTML element does not have a closing tag. The reading system must produce an error. (1) ❐
